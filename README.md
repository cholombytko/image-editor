# Image Editor

This is repository for Kyrylo's Cholombytko project - "web-application to create and edit images"

Need preinstalled:
1) Docker, docker-compose
2) node

For startup you need follow these steps:
1) Start backend part
```bash
docker-compose up
```

2) Start frontend app
```bash
cd frontend/

npm i

npm run dev
```
